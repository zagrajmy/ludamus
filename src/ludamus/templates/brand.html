  • {% translate "Coral marks the tabletop and small accents. It is never a surface: the logo does not appear on or as a full-coral field, so overlap with other orange brands is not a concern." %}
  • {% translate "Seats and wordmark inherit the text color; use the mono variant on coral-adjacent or busy grounds." %}

usage
  • {% translate "Use the full lockup where the platform speaks: the navbar, sign-in, e-mails, social cards." %}
  • {% translate "Use the bare mark where space is tight or the name is already on screen: favicons, avatars, badges." %}
  • {% translate "The logo signs the platform. Events and spheres speak under their own names." %}
  • {% translate "Keep one chair of clear space around the mark, and never render it below 16 px or the lockup below 120 px wide." %}
don'ts
  • {% translate "Do not recolor. Coral, ink, and mono are the whole palette." %}
  • {% translate "Do not stretch, rotate, or crop the mark." %}
  • {% translate "Do not add shadows, gradients, or outlines." %}
  • {% translate "Do not redraw the mark or set the wordmark in another font. Copy the templates." %}
  • {% translate "Do not place the logo on busy grounds. Switch to mono or calm the background." %}
  • {% translate "Do not build the logo into another logo, product name, or event brand." %}
