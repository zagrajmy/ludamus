# Filter spaces in the agenda builder

As a sphere manager, I want to narrow the agenda builder grid to spaces
matching a text query, so that an event with many spaces stays usable.

As a sphere manager, I want the filter to match against both area name
and space name, so that I can narrow by either the venue layout or the
specific room.

As a sphere manager, I want clearing the filter to restore the full
grid, so that filtering is non-destructive.
