# Agenda builder session search

As an organiser, I want the session search to match sessions by title,
display name, or any of their track names, so that I can fish a session
out of the global list without applying other filters first.

As an organiser, I want every term in a multi-word query to match
somewhere in a session, so that adding terms narrows the results
instead of expanding them.
