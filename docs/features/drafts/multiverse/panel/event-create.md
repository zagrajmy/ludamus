# Create event

As a sphere manager, I want to create an event without administrator
access, so that I can set up a new edition myself.

As a sphere manager, I want to land on the new event after creating it,
so that I can start configuring it immediately.

As a sphere manager, I want to seed a new event from an existing one's
structural configuration, so that I do not recreate venues, fields, and
categories by hand for every edition.

As a sphere manager, I want time-bound entities to shift relative to
the new event's start when cloning, so that a cloned schedule lines up
with the new dates.

As a sphere manager, I want cloning to copy structure but not the
previous edition's submissions and sessions, so that I start with a
clean slate of content.
