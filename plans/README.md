## Audit summary

Four parallel read-only audits (correctness+security, perf+tests,
tech-debt+deps, DX+docs+direction) over the whole `src/` tree at
standard depth. Every finding below was re-verified against the cited
code by the advisor before making this list. Not audited: vendored
frontend deps, `static/vendor`, migrations, locale catalogs, and the
`docker/` + deploy workflow internals beyond what CI wiring required.

Notable clean bill of health: MCP token auth (signed, constant-time,
per-request standing re-check), panel object-scope authorization
(matches `docs/refactors/panel-object-scope-authz.md`), enrollment
capacity locking (`select_for_update` + re-check under lock), nh3
markdown sanitization, image-upload validation, and overall test
posture (96% patch gate, mutation testing, Django-free unit layer).

  ~737 LOC): the documented next migration
  (`docs/refactors/glimpse-strangler.md`), L effort, needs

  maintainer choosing scope.
